# Hermes Funds — 需求文档 v1.1

> 项目：个人/家庭资金管理系统
> 版本：v1.1（已确认）
> 阶段：需求分析 ✓ 已确认
> 日期：2025-05-03

---

## 产品愿景

> 用日常语言告诉 Hermes Funds 你花了什么、赚了多少钱、投资了什么，它自动帮你整理成清晰的账本和投资收益分析。

---

## 一、核心原则（优先级排序）

| 优先级 | 模块 | 说明 |
|--------|------|------|
| **P0** | **记账核心** | 收支记录、自然语言解析、分类管理、账户管理、报表 |
| **P1** | **投资组合** | 基金/股票/ETF 持仓、净值查询、收益计算、定投计划 |

> ✅ P0 功能完成后系统即可投入使用。
> P1 功能可在 P0 稳定后陆续接入。

---

## 二、投资品种覆盖（尽可能全）

| 类型 | 示例 | 数据源 |
|------|------|--------|
| 场外基金 | 招商中证白酒、兴全合润、指数基金 | 东方财富 fundgz API |
| 股票 | 贵州茅台、腾讯控股 | 新浪/腾讯行情 API |
| ETF | 沪深300ETF、纳指ETF | 新浪/腾讯行情 API |
| 黄金 | 纸黄金、黄金ETF | 腾讯行情 API |
| 银行存款 | 活期/定期 | 手动录入 |
| 货币基金 | 余额宝、零钱通 | 手动录入（收益手动计算） |
| 可转债 | — | 新浪行情（待验证） |
| REITs | — | 手动录入（无实时行情） |
| 私募/信托 | — | 手动录入 |

> 📌 实时行情覆盖：基金、股票、ETF、黄金
> 📌 手动维护：存款、货币基金（无实时行情）、私募等

---

## 三、数据源

| 数据 | 接口 | 费用 |
|------|------|------|
| 基金净值/估值 | `fundgz.1234567.com.cn` | ✅ 免费，无需 Key |
| 股票实时行情 | `hq.sinajs.cn`（新浪） | ✅ 免费，无需 Key |
| 股票实时行情 | `qt.gtimg.cn`（腾讯） | ✅ 免费，无需 Key |

> ⚠️ 均为非官方接口，需做缓存（建议 5-10 分钟） + 降级（失败时用上次缓存值）
> ⚠️ 仅供个人/家庭内使用

## 四、P0 功能（记账核心）

### M1：自然语言记账引擎

| ID | 功能 | 描述 |
|----|------|------|
| NL-1 | 意图识别 | 区分：支出/收入/查询 |
| NL-2 | 金额提取 | 支持"100元""50块""200.5元" |
| NL-3 | 日期解析 | 支持"今天""昨天""上周三""5月1日" |
| NL-4 | 分类推断 | 根据描述推断分类（餐饮→餐饮，工资→工资收入） |
| NL-5 | 歧义确认 | 信息不足时主动提问 |

**对话示例：**
```
用户: 餐饮花费100元    → 支出 ¥100，分类：餐饮，日期：今天
用户: 昨天买书花了50块 → 支出 ¥50，分类：书籍，日期：昨天
用户: 工资收入2000元   → 收入 ¥2000，分类：工资，日期：今天
用户: 超市156.5元      → 支出 ¥156.5，分类：日用，日期：今天
```

### M2：账户管理

| ID | 功能 |
|----|------|
| AC-1 | 账户 CRUD（现金/银行卡/支付宝/微信/投资账户） |
| AC-2 | 余额初始化 |

### M3：收支记录

| ID | 功能 |
|----|------|
| TX-1 | 收支记录 CRUD |
| TX-2 | 支持按日期/分类/账户筛选 |
| TX-3 | 编辑/删除历史记录 |

### M4：分类管理

| ID | 功能 |
|----|------|
| CAT-1 | 支出/收入二级分类体系（支持自定义） |
| CAT-2 | 预设常用分类（餐饮/交通/购物/工资/奖金等） |

### M5：报表（P0 范围）

| ID | 功能 |
|----|------|
| RP-1 | 月度收支趋势图（柱状图） |
| RP-2 | 分类占比饼图 |
| RP-3 | 当月收支汇总卡片 |

## 五、P1 功能（投资组合）

### M6：持仓管理

| ID | 功能 | 说明 |
|----|------|------|
| INV-1 | 基金/股票/ETF/黄金 买入记录 | 支持按金额或份额录入 |
| INV-2 | 卖出/赎回记录 | 自动关联持仓 |
| INV-3 | 分红入账 | 份额增加或现金入账 |
| INV-4 | 持仓列表 | 展示当前所有投资持仓 |

### M7：基金净值

| ID | 功能 | 说明 |
|----|------|------|
| NAV-1 | 按基金名称/代码搜索 | 东方财富接口 |
| NAV-2 | 实时 NAV 查询 | fundgz API，缓存 5 分钟 |
| NAV-3 | 批量持仓净值更新 | 每次打开或定时刷新 |

### M8：收益计算

|----|------|

| PROF-2 | 总收益 / 收益率 |
| PROF-3 | 日涨跌幅（基金估算值/股票实时价） |
| PROF-4 | 持仓组合汇总（总投入/总现值/总收益） |

### M9：定投计划

| ID | 功能 |
|----|------|
| DIP-1 | 设置周/月定投计划 |
| DIP-2 | 定投记录自动生成交易 |
| DIP-3 | 定投收益率展示 |

### M10：投资报表

| ID | 功能 |
|----|------|
| INV-RP1 | 持仓收益明细 |
| INV-RP2 | 历史买卖记录 |
| INV-RP3 | 组合资产配置图 |

## 七、对话设计

### 7.1 记账命令关键词

| 类型 | 触发词 |
|------|--------|
| 支出 | 花、买、花费、消费、支出、付、扣除 |
| 收入 | 收、到账、收入、工资、奖金、分红、利息 |
| 查询 | 多少、花了多少钱、收支、收益怎么样 |

### 7.2 投资命令关键词

| 类型 | 触发词 |
|------|--------|
| 买入 | 买入、购买、定投、投资（基金/股票） |
| 卖出 | 卖出、赎回、减仓 |
| 查询 | 持仓、收益、净值、行情 |

### 7.3 典型对话

```
支出：
用户: 餐饮花费100元
系统: ✅ 已记录
      💸 支出 ¥100.00 | 餐饮 | 今天

收入：
用户: 工资收入2000元
系统: ✅ 已记录
      💰 收入 ¥2,000.00 | 工资 | 今天

投资买入：
用户: 买入1000元招商中证白酒
系统: ✅ 已记录
      📈 买入 招商中证白酒(000011) ¥1,000.00 | 今天

投资查询：
用户: 我的投资收益怎么样
系统: 📊 投资收益概览
      总投入：¥50,000 | 当前价值：¥53,200 | 收益：¥3,200 (+6.4%)

查询收支：
用户: 这个月花了多少
系统: 📊 2025年5月
      💸 支出：¥3,456
      💰 收入：¥8,000
      📈 结余：¥4,544
```
