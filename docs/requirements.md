# Hermes Funds — 需求文档 v1.0

> 项目：个人/家庭资金管理系统
> 版本：v1.0（草稿）
> 阶段：需求分析
> 日期：2025-05-03

---

## 1. 产品愿景

**一句话描述：**
> 用日常语言告诉 Hermes Funds 你花了什么、赚了多少钱、投资了什么，它自动帮你整理成清晰的账本和投资收益分析。

**核心价值：**
- 🔄 **零门槛记账** — 说话就能记账，不需要打开 App 手动填表单
- 📈 **投资全覆盖** — 基金/股票/ETF 买入/卖出/定投全追踪，实时收益
- 🏠 **家庭友好** — 支持多成员，各自有账本，汇总看全家财务
- 🔒 **数据自主** — 本地优先，开源透明

## 2. 用户场景

### 2.1 日常记账

| 用户说（示例） | 系统自动解析 | 记录内容 |
|---------------|------------|---------|
| 餐饮花费100元 | 支出 | 分类：餐饮，金额：100，日期：今天 |
| 昨天买书花了50块 | 支出 | 分类：教育/书籍，金额：50，日期：昨天 |
| 工资收入2000元 | 收入 | 分类：工资，金额：2000，日期：今天 |
| 房租支出3000 | 支出 | 分类：房租，金额：3000，日期：今天 |
| 超市买东西用了156.5元 | 支出 | 分类：日用，金额：156.5，日期：今天 |
| 收到奖金500元 | 收入 | 分类：奖金，金额：500，日期：今天 |

### 2.2 投资管理

| 用户说（示例） | 系统自动解析 | 记录内容 |
|---------------|------------|---------|
| 投资基金1000元 | 投资买入 | 基金名：待确认，金额：1000，定投日：今天 |
| 买入500元沪深300基金 | 投资买入 | 基金名：沪深300，金额：500 |
| 卖出基金200元 | 投资卖出 | 基金名：待确认，金额：200（需确认份额） |
| 每周一定投200元 | 定投计划 | 频率：每周一，金额：200 |
| 基金分红50元 | 分红入账 | 分红金额：50 |

### 2.3 查询/统计

| 用户说（示例） | 系统行为 |
|---------------|---------|
| 这个月花了多少 | 返回本月支出总额 + 分类明细 |
| 我的投资收益怎么样 | 返回各持仓收益/收益率 |
| 上周收支情况 | 返回上周收入/支出/结余 |
| 帮我看看财务状况 | 返回资产负债表概览 |

## 4. 对话交互设计

### 4.1 典型对话流

**场景：用户说"餐饮花费100元"**

```
用户: 餐饮花费100元
系统: ✅ 已记录
      📅 2025-05-03
      💸 支出 ¥100.00
      🏷️ 餐饮
      📂 默认账户
      [修改]
```

**场景：信息不足**

```
用户: 投资基金
系统: 🤔 请告诉我以下信息：
      1. 基金名称或代码？
      2. 投资金额？
      [直接回复即可，例如：招商中证白酒指数，500元]
```

**场景：查询**

```
用户: 这个月花了多少钱
系统: 📊 2025年5月 收支概览
      💸 支出：¥3,456.00
      💰 收入：¥8,000.00
      📈 结余：¥4,544.00

      Top 支出分类：
      1. 餐饮 ¥856.00 (24.8%)
      2. 交通 ¥420.00 (12.1%)
      3. 购物 ¥680.00 (19.7%)
```

### 4.2 命令关键词

| 类型 | 触发词示例 |
|------|-----------|
| 支出 | 花、买、花费、消费、支出、付、扣除 |
| 收入 | 收、到账、收入、工资、奖金、分红、利息 |
| 投资买入 | 买入、购买、定投、投资（基金/股票） |
| 投资卖出 | 卖出、赎回、减仓 |
| 查询 | 多少、花了多少钱、收支、收益怎么样 |

## 6. 数据范围

### 6.1 资产类型

| 类型 | 示例 | 行情数据 |
|------|------|---------|
| 场外基金 | 招商中证白酒、兴全合润 | 天天基金网 API |
| 股票 | 贵州茅台、腾讯控股 | 实时行情 |
| ETF | 沪深300ETF、纳指ETF | 实时行情 |
| 银行存款 | 活期/定期 | 手动录入 |
| 货币基金 | 余额宝 | 万份收益 |
| 黄金 | 纸黄金/黄金ETF | 实时行情 |

### 6.2 投资相关计算

```
持仓成本 = Σ(买入金额) - Σ(卖出金额 + 分红)
单位成本 = 持仓成本 / 当前持有份额
总收益 = (当前净值 - 成本均价) × 持有份额
收益率 = 总收益 / 持仓成本 × 100%
日涨跌幅 = (今日净值 - 昨日净值) / 昨日净值 × 100%
```

## 7. 边界/排除范围

### 7.1 本版本不包含（v1.0）

- ❌ 信用卡还款管理
- ❌ 借贷/债务管理
- ❌ 保险管理
- ❌ 多币种自动汇率转换
- ❌ 银行账单自动同步
- ❌ 税务申报功能

### 7.2 假设条件

- 用户位于中国，货币为人民币（CNY）
- 基金以"万元"场外基金为主
- 投资数据通过手动维护 + 第三方行情 API
